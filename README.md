# 网站图片上传助手

Tampermonkey 油猴脚本，用于电商网站商品图片的批量上传。

## 功能

### 详情图上传
- 图片统一缩放到 850px 宽度（支持小图放大）
- 自动叠加分平台 Logo 水印（左下角）
- 自动注入 Alt / Title 属性（SEO 优化）
- 并发上传控制（默认最多 3 张同时上传）
- 输出格式 JPEG，质量 100%

### 主图上传（V3.2 新增）
- 通过 XHR 层 + iframe 监听自动拦截主图上传，上传前自动处理
- 强制输出 500×500 正方形图片（白色背景填充，图片居中）
- JPEG 质量 90%（体积小，上传快）
- 自动叠加分平台 Logo 水印（左下角）
- 处理期间显示遮罩提示

### 本地保存 / 高清归档（V3.7 新增，两个版本同步支持）
- 水印处理流程中可同时保存一张**原图分辨率 + Logo 水印**的高清版到本地（浏览器默认下载目录）
- 两个版本行为不同:
  - **Tampermonkey 版**:默认**关闭**,通过菜单 **"切换本地保存 (开/关)"** 手动开启,`GM_setValue` 持久化(top 与 iframe 共享)
  - **Chrome 插件版**:**默认一直开启**,无开关按钮,装了插件就一直归档
- **本地归档 ≠ 上传到网站的版本**：
  - 上传到网站：压缩版（详情图 850 宽、主图 500 正方形），保证网站加载速度
  - 本地归档：原分辨率 + 水印，保留原始长宽比（主图也不强制正方形），JPEG 质量 0.95，适合作为素材备份
- 文件名格式：`{平台前缀}{原文件名}-orig.jpg`，例如 `SZ_product123-orig.jpg` / `GD_xxx-orig.jpg`
- 对详情图 + 主图两条链路均生效

## 历史版本说明
- **V3.7**：新增水印后本地保存功能（菜单开关，默认关）
- **V3.5**：Logo 预存为 base64 嵌入脚本，不再依赖 Tampermonkey 存储，换浏览器/清缓存不丢失
- **V3.4**：主图水印比例调整为 0.35（原 0.25），使主图 logo 大小与详情图视觉协调
- **V3.3**：自动识别详情图对话框（URL 含 `save=_Detail` / `Description`），跳过主图处理，避免详情图被误处理为正方形

## 支持平台

- GundamIT (`gundamit.com`)
- ShowzStore (`showzstore.com`)

## 安装

1. 安装 [Tampermonkey](https://www.tampermonkey.net/) 浏览器扩展
2. 将 `图片上传脚本.user.js` 导入 Tampermonkey

## 文件说明

| 文件 | 说明 |
|------|------|
| `图片上传脚本.user.js` | Tampermonkey 当前版本（V3.7） |
| `图片上传脚本-v1.5-初始版.user.js` | 初始版本存档 |
| `chrome-extension/` | Chrome 插件版（V3.7，功能与 Tampermonkey 对齐） |
