# 网站图片上传

## 项目概述

Tampermonkey 油猴脚本，用于 SZ (ShowZ Store) 和 GD (GundamIT) 电商网站后台的商品图片批量上传。

## 核心文件

| 文件 | 说明 |
|------|------|
| `图片上传脚本.user.js` | 当前版本（V3.6） |
| `图片上传脚本-v1.5-初始版.user.js` | 初始版本存档 |

## 功能

### 详情图上传
- 图片统一缩放到 850px 宽度（支持小图放大）
- 自动叠加分平台 Logo 水印（左下角）
- 自动注入 Alt / Title 属性（SEO 优化）
- 并发上传控制（默认最多 3 张同时上传）
- 输出格式 JPEG，质量 100%

### 主图上传（V3.2 新增）
- 通过 XHR 层 + iframe 监听自动拦截主图上传，上传前自动处理
- 强制输出 500×500 正方形图片（白色背景填充，图片居中）
- JPEG 质量 90%（体积小，上传快）
- 自动叠加分平台 Logo 水印（左下角）
- 处理期间显示遮罩提示
- **V3.3**：自动识别详情图对话框，跳过主图处理
- **V3.6**：Logo 改为 @resource 从 GitHub 加载（原图），移除内嵌 base64；新增 Chrome 插件版本
- **V3.5**：Logo 预存为 base64 嵌入脚本（400px 宽 PNG），不再依赖 GM 存储，换浏览器/清缓存不丢失
- **V3.4**：主图水印比例从 0.25 调整为 0.35，解决主图 logo 明显小于详情图的问题

## 支持平台

- GundamIT (`gundamit.com`)
- ShowzStore (`showzstore.com`)
